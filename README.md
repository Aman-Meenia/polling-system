# Tech-Stack used

```
NextJs, Shadcn, Postgres, Prisma, recharts, Docker, Zod
```

# How to run the app

### Add this to your `.env` file

```
DATABASE_URL="postgresql://admin:root@localhost:5432/test_db"
```

###

## Run Nodejs app

```
npm install
```

```
npm run build
```

```
npm run start
```

## Run Docker

```
docker-compose up --build -d
```
